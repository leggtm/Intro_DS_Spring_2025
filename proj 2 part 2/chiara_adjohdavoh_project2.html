Cleaning the Data

Next, the data must be cleaned and organized. I noticed that there were often multiple entires per year per country. For example, in the year 2000, South Africa had 7 entries. If each entry had a month associated with them, that might be useful, but because we do not know when in the year the data was collected, I wanted to consolidate the information so there was only one entry, per country per year. I started by using the groupby feature to organize first by year, then by country. Then, the rows were aggregated based on either mean or max, depending on what was more appropriate per column. For example, it was best to average it for the Average Temperature, but it was better to find the max for Sea Level Rise since that was likely taken later in the year. This grouping also sorts the data by year so the entries go from oldest to most recent.

Feature Engineering

Now that the data was a bit more consolidated, I decided to add some features. Firstly, a variable called "Abv_Avg_Events" returned either a 0 or 1 depending on if the entry had an above or below median number of extreme weather events. This felt useful for putting the number of extreme weather events in context.

Next, I created a categorical "climate" variable. Depending on the range the average temperature fell in, it would classify the climate as warm, temperate or cold. This could be useful for quickly determining if the average temperature in a country has changed signficantly.

Lastly, I added a categorical "region" variable. It simply filters based on the continent the country is on. This is a really helpful groupby variable.

In[526]:
df_new = df_grouped.copy() # copies the df to a new dataframe

df_new['Abv_Avg_Events'] = (df_new['Extreme_Weather_Events'] >= df_new['Extreme_Weather_Events'].median()).astype(int) # works like a boolean that is 1 if the number of events is greater than the median

def climate(temp): # simple if else for classifying the climate based on temp
    if temp >= 22:
        return "Warm"
    elif temp >= 12:
        return "Temperate"
    else:
        return "Cold"
df_new['Climate'] = df_new['Avg_Temperature_degC'].apply(climate) 

def region(country): # takes the string and sorts it using the if else, returning the region. this is not feasible for larger sets but it worked fine for this set of only 14 countries
    if country == "UK" or country == "Germany" or country == "France" or country == "Russia":
        return "Europe"
    elif country == "'South Africa'":
        return "Africa"
    elif country == "Australia":
        return "Oceania"
    elif country == "Mexico" or country == "Argentina" or country == "Brazil":
        return "South America"
    elif country == "Canada" or country == "USA":
        return "North America"
    elif country == "India" or country == "Japan" or country == "Indonesia" or country == "China":
        return "Asia"
df_new['Region'] = df_new['Country'].apply(region)

Correlation matrix

This is a correlation matrix of all the numerical variables. There is a very strong correlation between extreme weather events and above average weather events but this is to be expected, as the latter is directly based on the former. Disregarding those two, the variable pairs that have the strongest correlations include:

  • Sea Level Rise and Rainfall
  • Sea Level Rise and Year
  • Sea Level Rise and Extreme Weather Events / Above Avg Events
  • Rainfall and Extreme Weather Events
  • Average Temperature and Renewable Energy Percentage (Inversely Correlated)
  • Population and Renewable Energy Percentage (Inversely Correlated)

A lot of these correlations make perfect sense, as our knowledge of climate change informs us that rainfall and the level of the sea are part of the same system that cause them to be related to one another, even if they don't directly relate. Sea level has also consistently been an indicator of other much larger issues with climate change, due to the effects that climate change has on our very delicate glaciers. Average temps and renewable energy being inversely, specifically, related makes perfect sense. The purpose of renewable energy is to free us from the effects of fossil fuels, which are known to release CO2 and cause the atmosphere to heat up. Population and renewable energy being inversely correlated is actually very surprising to me. It's interesting that countries that are larger actually tend to have less renewable energy.
